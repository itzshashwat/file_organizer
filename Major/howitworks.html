How It Works?

What can you do from this tool

In order to provide an efficient method to store and locate files.

Help

Help Function tells the user about the steps of implementation of organize and tree function.

Organize

Organize Function done the job for user to organize the unsorted files in a sorted manner in the directory of your system.

Tree

Tree Function done the job for user to print the sorted files in a tree structure format for better understaning to the location of the files in each folder.

How to use File Organizer

This shows a required command for using a given three functionality and how it will it appear.

Help Command

This command helps the user about the steps of implementation of organize and tree function. The format of firing help command in cmd/terminal is "FO help"

Organize Command

Organize Function is used to organize the unsorted files in a sorted manner in the directory. The format of firing help command in cmd/terminal is "FO organize 'directory_path' ".

Tree command

Tree Function is used to print the sorted files in a tree structure format for better understaning to the location of the files in each folder. The format of firing help command in cmd/terminal is "FO tree 'directory_path' ".
