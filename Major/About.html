About Us

Why did we come up with this tool...

There is never enough time to do everything, but there is always enough time to do the most important thing.”
— Brian Tracy

Have you ever found yourself struggling to keep track of files with different formats and extensions while working on a project? It can be frustrating and time-consuming to manually manage and organize files. That's where a customized file organizer tool comes in handy.

By using the extension of each file as a guide, this tool can automatically sort files into appropriate folders, saving you time and effort. Not only does it make file management more convenient, but it also improves the overall organization of your computer system. With related files kept together in designated folders, you can quickly and easily find what you need.

This tool is command line-based, giving you more control and efficiency in navigating your files and directories. No more wasted time searching for files or manually moving them around. This exciting and efficient tool streamlines your workflow and takes the hassle out of file management.
