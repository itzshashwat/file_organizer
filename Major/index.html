What can you do from this tool

In order to provide an efficient method to store and locate files.

Help

Help Function tells the user about the steps of implementation of organize and tree function.

Organize

Organize Function is used by the user to organize the unsorted files in a sorted manner in the directory of your system.

Tree

Tree Function is used by user to print the sorted files in a tree structure format for better understaning to the location of the files in each folder.

How to use File Organizer

So that's how the requisted command will appear and perform.

Help Command

This is the first and initial command tells the user about the steps of implementation of organize and tree function. The format of firing a command in cmd i.e " node FO.js _help_ "

Organize Command

Organize Function done the job for user to organize the unsorted files in a sorted manner in the directory of your system. For using this function User needs to fire a command i.e " node FO.js organize _directory name_ ".

Tree command

Tree Function done the job for user to print the sorted files in a tree structure format for better understaning to the location of the files in each folder. for using this tree function user needs to fire a command in cmd with thire directory name in which user wants the tree structure i.e "node FO.js tree _directory name_".
